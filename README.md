# SwitchMaster

SwitchMaster est une application Electron moderne permettant de gérer et basculer facilement entre plusieurs comptes Riot Games (League of Legends, Valorant).

## Fonctionnalités

*   **Gestion de comptes multiples** : Ajoutez, modifiez et supprimez vos comptes.
*   **Connexion automatique** : Basculez entre les comptes en un clic sans retaper vos identifiants.
*   **Statistiques en direct** : Visualisez votre rang et niveau actuel (LoL & Valorant).
*   **Protection par Code PIN** : Sécurisez l'accès à l'application.
*   **Interface Moderne** : Design sombre, épuré et réactif.
*   **Mises à jour automatiques** : L'application vous notifie des nouvelles versions.

## Installation

1.  Téléchargez la dernière version depuis la page [Releases](https://github.com/votre-user/SwitchMaster-electron/releases).
2.  Installez l'application via le fichier `.exe`.
3.  Lancez SwitchMaster.

## Sécurité

Les mots de passe sont stockés de manière chiffrée localement sur votre pc
